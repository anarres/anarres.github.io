Simple fundraising thermometer


To save the image, right-click on it and select "Save image As"

Advantages: you don't need to install anything, just use this website to generate an image, then post that image to your blog / website / social networking page.

Disadvantages: it doesn't work automatically; everytime you want to increase the 'temperature' level on the thermometer you have to remove the old image from your website, and generate a new image to replace it.
